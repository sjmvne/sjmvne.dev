# sjmvne.dev

Portfolio personale e CV digitale di Simone Pepe.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Motion
- Mapbox

## Sviluppo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm lint
pnpm typecheck
pnpm build
```

## Deploy

Il progetto e pensato per deploy su Vercel.
